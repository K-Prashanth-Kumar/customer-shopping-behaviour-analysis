# SQL

This folder contains the main analysis of the project.

After preparing the data, SQL was used to answer business questions and extract insights about customer behavior and revenue patterns.

### Work Done

- Calculated total revenue by gender, age group, and season  
- Analyzed customer segments and spending behavior  
- Evaluated how discounts affect sales  
- Identified top products and categories  
- Analyzed location-wise performance  
- Studied payment methods and shipping preferences  

### Key Questions Answered

- Which customers generate the most revenue?  
- Which products and categories perform best?  
- Does season affect sales?  
- How do discounts impact revenue?  
- Which locations are most profitable?  

### Why This Step Matters

SQL helps in:
- Converting data into meaningful insights  
- Understanding business performance  
- Supporting data-driven decisions  

### Outcome

- Clear understanding of customer behavior  
- Identification of key revenue drivers  
- Insights used for dashboard and report
