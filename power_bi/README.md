# Power BI

This folder contains the interactive dashboard created from the analysis.

The goal was to convert SQL insights into visual format so that stakeholders can easily understand and explore the data.

### Work Done

- Created KPIs: total revenue, customers, orders, average values  
- Built charts for category performance and age groups  
- Visualized discount impact and customer behavior  
- Added location-based map for geographic analysis  
- Used slicers for filtering (gender, season, location, etc.)  

### Why This Step Matters

Raw numbers are hard to understand. Visualization helps:
- Quickly identify patterns  
- Compare performance across segments  
- Support decision-making  

### Outcome

- Interactive dashboard  
- Easy-to-understand business insights  
- Better data storytelling
